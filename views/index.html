API Project: URL Shortener Microservice

User Story:
  • I can POST a URL to [project_url]/api/shorturl/new and I will receive a shortened URL in the JSON response. Example : {"original_url":"www.google.com","short_url":1}
  • If I pass an invalid URL that doesn't follow the http(s)://www.example.com format, the JSON response will contain an error like {"error":"invalid URL"}
  • When I visit the shortened URL, it will redirect me to my original link.
Short URL Creation
example: POST [project_url]/api/shorturl/new


to get redirected to the short url:

follow the following pattern:
{thisURL}/api/short-url/< shortURL value >
